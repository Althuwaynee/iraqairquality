Methodology

How we collect, process, and analyze air quality data

1
Data Collection

Satellite data from NASA MODIS and VIIRS, combined with ground monitoring stations across Iraq.

2
Processing

Calibration using ground-truth measurements and atmospheric correction algorithms.

3
Analysis

Statistical analysis and machine learning models to identify trends and patterns.

4
Visualization

Interactive maps and charts using D3.js and Leaflet for clear communication.
