# Iraq Air Quality Monitoring Website

A professional, interactive website for monitoring and visualizing air quality data across Iraq.

## Features
- Real-time PM2.5 data visualization
- Interactive insights and trend analysis
- Responsive design for all devices
- Bilingual support (English/Arabic)
- Open data access for researchers
